ЛанаБанк — графическое приложение банковской системы на Python (Tkinter).
Приложение предоставляет функционал для регистрации пользователей, авторизации, проведения переводов по номеру телефона, оплаты услуг, просмотра баланса и истории операций. Интерфейс оформлен в тёмно-синих тонах, кнопки — бирюзовые. Внутренняя база пользователей реализована в виде словаря Python.

Основные функции:
Регистрация и вход пользователей
Проверка баланса (баланс по умолчанию — 200 000 ₽)
Перевод средств по номеру телефона
Добавление новых пользователей
Оплата услуг
История платежей
Возврат в предыдущее меню # BankSistem
